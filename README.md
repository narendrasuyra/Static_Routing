# Static Routing in Cisco Packet Tracer

## Project Overview

This project demonstrates the implementation of Static Routing in Cisco Packet Tracer using a multi-site network topology. The network consists of three routers, three switches, client PCs, and servers connected through LAN and WAN segments. Static routes are manually configured on each router to enable communication between remote networks.

The project showcases network design, IP addressing, router configuration, routing table verification, connectivity testing, and troubleshooting techniques commonly used in enterprise networking environments.

## Project Files

* **Static_Routing.pkt** – Cisco Packet Tracer simulation file containing the complete network topology and configuration.
* **Static_Routing_Documentation.docx** – Detailed project documentation including topology design, IP addressing scheme, router configurations, verification steps, screenshots, and troubleshooting notes.
